# ResimGalerisi
- Modern ve responsive (mobil uyumlu) galeri tasarımı  
- Resim üzerine gelince büyüme ve kararma efekti  
- Resmin üzerinde beliren yarı saydam açıklama yazıları  
- Fotoğrafa tıklayınca büyük boyutta açılan modal pencere  
- Modal içinde **Next / Prev** butonları ile slayt şeklinde gezinebilme  
- Açıklamalı ve temiz JavaScript yapısı  
- Kolay düzenlenebilir HTML + CSS + JS yapısı
- Güncellemeler devam edecek.
